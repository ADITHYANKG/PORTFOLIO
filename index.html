SKILL

HTML

HTML, short for Hypertext Markup Language, is the standard language used to create web pages. It uses tags to structure the content and layout of a webpage. These tags define elements such as headings, paragraphs, images, links, and more. HTML is essential for building the basic structure of a website.

CSS

CSS, or Cascading Style Sheets, styles the appearance of HTML document. It defines colors, fonts, layout, and more, enhancing webpage aesthetics and user experience. CSS separates content from presentation, allowing for consistent and responsive design across different devices and screen sizes.

JavaScript

JavaScript (JS) is a versatile programming language primarily used for adding interactivity to web pages. It enables dynamic behavior like animations, user input validation, and updating content without reloading the page. JS works alongside HTML and CSS, making web pages more interactive and responsive for users.

Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It was created by Guido van Rossum and first released in 1991. Python is designed to be easy to learn and use, with a clean syntax that emphasizes readability, which makes it a popular choice for beginners as well as experienced developers.

C programming

C is a general-purpose, high-level programming language known for its efficiency and control over system resources. It allows low-level memory manipulation through pointers, making it suitable for system-level programming. C is structured, enabling developers to organize code into smaller, manageable functions.

Embedded System

An embedded system is a specialized computing system that is designed to perform dedicated functions or tasks within a larger system. It typically consists of hardware (such as microcontrollers or microprocessors) and software (firmware) working together to control devices or processes..
